# 用户手册

## REPL 命令

| 命令 | 说明 |
|------|------|
| `help` | 显示帮助信息 |
| `tables` | 列出所有表 |
| `schema <name>` | 查看表结构 |
| `exit` / `quit` | 退出 REPL |

## 支持的数据类型

| 类型 | 说明 | 示例 |
|------|------|------|
| INT | 整数 | 42 |
| FLOAT | 浮点数 | 3.14 |
| TEXT | 文本字符串 | 'hello' |
| BOOL | 布尔值 | TRUE / FALSE |
| NULL | 空值 | NULL |

## 支持的操作符

| 类型 | 操作符 |
|------|--------|
| 比较 | =, <>, <, >, <=, >= |
| 逻辑 | AND, OR, NOT |
| 算术 | +, -, *, / |
